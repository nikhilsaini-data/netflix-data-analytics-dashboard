# Netflix Data Analytics Dashboard

This project is my first data analytics dashboard created using Microsoft Excel.  
As a beginner in data analytics, this project helped me understand the complete workflow of data cleaning, analysis, visualization, and dashboard design using a real-world dataset.

The goal of this project was to analyze Netflix content data and turn raw data into meaningful insights through data visualization and storytelling.

---

## Project Overview

In this project, I worked with a Netflix dataset containing movies and TV shows information such as:
- Title
- Genre
- Country
- Release Year
- Rating
- Duration
- Directors

Using Excel, I cleaned the dataset, handled missing values, created Pivot Tables, and designed an interactive dashboard to identify trends and patterns in Netflix content.

---

## Tools & Skills Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- KPI Cards
- Dashboard Design
- Data Visualization

---

## Key Insights

- Movies contributed around 69.62% of total Netflix content.
- 2018 had the highest number of title releases.
- Drama was the most common genre on Netflix.
- TV-MA was the most frequent content rating.
- United States produced the highest number of titles.
- The longest movie duration in the dataset was 312 minutes.
